Desktop · ícones · clique direito · organização

1
Olhe para a área de trabalho e identifique pelo menos 3 ícones. Anote os nomes abaixo.
Ícones são os desenhos na tela. Cada um abre um programa ou pasta.
Quais ícones você encontrou?
2
Clique com o botão direito em uma área vazia da área de trabalho e explore o menu. Conte quantas opções aparecem.
O botão direito sempre mostra opções relacionadas ao que você clicou.
Quantas opções apareceram no menu?
3
Crie uma nova pasta na área de trabalho: botão direito → "Novo" → "Pasta". Nomeie como Exercício Windows.
Pastas servem para organizar seus arquivos.
4
Renomeie a pasta: botão direito nela → "Renomear" → escreva Aula de Informática → Enter.
Também é possível clicar devagar duas vezes no nome para renomear.
5
Use o atalho Win + D para ocultar todas as janelas e ver a área de trabalho. Pressione novamente para voltar.⊞+D
Muito útil quando você tem muitas janelas abertas.
O que aconteceu ao pressionar Win + D?
📌 Dica: A área de trabalho é como sua mesa de trabalho virtual. Use pastas para manter tudo organizado!

Explorador de Arquivos
File Explorer · pastas · copiar · mover · organizar
0/6 ▼
1
Abra o Explorador de Arquivos com o atalho Win + E.⊞+E
Você também pode clicar no ícone de pasta amarela na barra de tarefas.
2
No painel esquerdo, navegue pelas pastas: Documentos, Downloads e Imagens. Clique em cada uma e veja o conteúdo.
O que você encontrou em cada pasta?
3
Vá em Documentos e crie uma pasta chamada Aula Prática (botão direito → Novo → Pasta).
4
Abra o Bloco de Notas, escreva Olá, Windows! e salve com Ctrl + S dentro da pasta Aula Prática.Ctrl+S
Na janela de salvar, navegue até Documentos → Aula Prática antes de clicar em Salvar.
5
Copie o arquivo: clique nele → Ctrl + C → vá para a Área de Trabalho → Ctrl + V.Ctrl+CCtrl+V
6
Clique na barra de endereço do Explorador e digite C:\ e pressione Enter. Veja o que aparece.
O que apareceu quando você digitou C:\ ?
📌 Estrutura típica: C: → Usuários → [seu nome] → Documentos / Downloads / Imagens / Área de Trabalho
